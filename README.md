# ticket-management

## Installation

télécharger le backend : https://github.com/cmoileboss/ticket-management

Dans le dossier backend :  
```pip install uvicorn```  
```pip install fastapi```  

Dans le dossier frontend :  
```npm install```

## Lancement de l'API
  
```cd backend```  
```./launch.bat```

## Lancement du frontend
  
```cd frontend```  
```npm run dev``` 

## Endpoints  

**Get Endpoints**  
http://localhost/tickets  
http://localhost/count-status

**POST Endpoints**  
http://localhost/tickets/{id}

**DELETE Endpoints**  
http://localhost/tickets/{id}

**PATCH Endpoints**  
http://localhost/tickets/{id}
